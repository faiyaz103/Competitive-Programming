# Modular Congruences
```
a & b congruent to each other under N if:
    a % N = b % N = r
Expressed as:
    a ≡ b (mod N)
Example:
    13 ≡ 41 (mod 7)
```
## Some Properties
```
if a % N = b % N = r
```
1. 
```
(a+x) % N = (b+x) % N
```
2. 
```
(a*x) % N = (b*x) % N
```
3. 
```
(a-b) % N = 0
```
4. If a*b = c
```
(a%N) * (b%N) = c%N
```
